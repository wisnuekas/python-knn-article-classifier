
1. install pyton 3. download di python.org
2. restart windows
3. check python di cmd dengan mengetik python -V
4. check pip di cmd dengan mengetik pip -V
5. install jupyter lab. 'pip install jupyterlab'
6. buka explorer. masuk ke folder project
7. buka cmd dengan mengetik 'cmd' di address bar
8. buka jupyterlab dengan command 'jupyter-lab'. browser akan terbuka dengan interface jupyter
9. buka jendela baru cmd
10. install package yang di butuhkan program. dengan cara 'pip install ntlk pandas numpy Sastrawi'. jika ada yang kurang, install package dengan pip install nama_package
11. tunggu pengunduhan  package sambil ngopi2
12. setelah selesai buka kembali jupyter
13. jalankan dengan menakan button run di cell 1 
14. jika cell berubah menjadi simbol * (bintang) dan kernel di pojok kanan atas/ pojok kiri bawah dengan keterangan idle menjadi busy, maka program sedang berjalan
15. jika sudah berjalan dengan benar. ulangi jalankan di cell selanjutnya sampai terakhir
